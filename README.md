# RideShare Web Application

## Overview
This collaborative academic web project was developed to promote campus-focused carpooling as a solution to reduce carbon emissions, ease traffic congestion, and improve parking availability among students and staff.

## Features
- Landing page
- Admin dashboard
- Reward Management
- Announcement Management
- User Management
- Staff Management
- Driver Management
- Location Management
- Chat Function
- User registration UI
- Driver registration UI
- Login system
- Responsive header and footer
- Navigation system
- Menu system
- Logout

## Technologies Used
- HTML
- CSS
- PHP
- JavaScript

## Installation / Running Locally
1. Download and install XAMPP
2. Import the MySQL database
3. Place the project folder inside the XAMPP `htdocs` directory
4. Start Apache and MySQL in XAMPP
5. Open the project through localhost in a browser
